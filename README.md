# dep_viz - инструмент визуализации графа зависимостей для менеджера пакетов

### Описание

Dep_viz позволяет выводить содержимое конфигурационного файла и проверяет корректность его составления

Выполнил: Панков Алексей Константинович, группа ИКБО-60-24

---
### Запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ha1fisch1337/dep_viz.git
   cd dep_viz
   ```

2. Запустите основной скрипт:
   ```bash
   python main.py
   ```
